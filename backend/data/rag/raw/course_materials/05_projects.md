# פרויקטים בקורס

הקורס כולל 7 פרויקטים מעשיים.

1. ניתוח דאטה ראשון מקצה לקצה
2. עיבוד Dataset אמיתי מ-Kaggle
3. חיזוי מחירי דירות
4. אופטימיזציה של מודל Machine Learning
5. חיזוי נטישת לקוחות
6. סגמנטציית לקוחות
7. Deploy של מודל Machine Learning באמצעות FastAPI

בנוסף קיים פרויקט גמר מקיף הכולל עבודה עצמאית ובניית Portfolio.
