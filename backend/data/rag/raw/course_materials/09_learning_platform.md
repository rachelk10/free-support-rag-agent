# אופן הלמידה

הקורס מועבר באמצעות סרטוני וידאו מוקלטים.

אין שיעורי זום חיים.

הלמידה מתבצעת בקצב אישי.

## גישה לתוכן

הגישה לקורס היא למשך 3 שנים.

## העלאת תכנים

פרק ראשון יעלה במועד פתיחת הקורס.

לאחר מכן יתווספו תכנים חדשים מדי שבוע.

כל תכני הקורס צפויים לעלות בתוך כחודשיים.

## עדכונים עתידיים

כל העדכונים העתידיים כלולים במסגרת הקורס.
