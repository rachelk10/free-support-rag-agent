# קורס Machine Learning

## תיאור כללי

קורס Machine Learning מקיף בעברית, המלמד את תחום למידת המכונה מהיסודות ועד לפרויקטים מעשיים.

הקורס מיועד למי שרוצה להיכנס לעולם ה-AI, להרחיב את הידע המקצועי שלו, לבנות תיק עבודות ולהכיר את הכלים והטכניקות המרכזיים הנהוגים בתעשייה.

הקורס משלב תיאוריה, תרגול ופרויקטים מעשיים המבוססים על דאטה אמיתי.

## מטרות הקורס

- הבנה מעמיקה של אלגוריתמי Machine Learning
- עבודה עם דאטה אמיתי
- בניית מודלים בפייתון
- הערכת ביצועי מודלים
- פיתוח חשיבה מקצועית לפתרון בעיות
- בניית תיק עבודות
- הכנה ללמידה מתקדמת בתחומי Deep Learning ו-Generative AI

## יתרונות מרכזיים

- קורס בעברית
- לימוד בקצב אישי
- גישה ל-3 שנים
- תכנים עדכניים ורלוונטיים לשוק
- דגש מעשי ופרויקטים
- מחיר נגיש
